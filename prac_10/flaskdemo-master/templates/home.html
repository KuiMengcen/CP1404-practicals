Welcome to the Flask + Wikipedia API App

Explore the power of Flask and the Wikipedia API through this web application!

What is Flask?

Flask is a lightweight and powerful web framework for Python. It allows developers to quickly build web applications with dynamic content. Its simplicity and flexibility make it an ideal choice for projects of any size.

What is the Wikipedia API?

The Wikipedia API provides a programmatic way to access content from Wikipedia. This includes searching for articles, retrieving summaries, and exploring links to full articles.

Features of This Application

  • Search for Wikipedia articles directly from the search page.
  • View brief summaries of topics and links to full articles on Wikipedia.
  • Experience the integration of Flask and Jinja templates for dynamic and user-friendly web pages.

Get Started

Use the navigation bar above to explore the application. Click on "Search" to begin searching Wikipedia, or visit the "About" page to learn more about this project.
